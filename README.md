This minecraft mod reworks the very beginning of the game through careful rebalancing, without adding much new content. Almost everything is adjusted vanilla behavior.

## Philosophy
The goal is to make the early game feel fair and meaningful again, while staying as close as possible to vanilla Minecraft. No big tech trees, no new dimensions, no complicated systems.

## Main Changes
- Wooden tools and weapons are now crafted using **logs** instead of planks
- Almost all tools and weapons before diamond require **2-4 String**
- Punching logs with bare hands is inefficient - only **30% change** to drop a log
- Wooden, Stone, Copper, Iron tools are intentionally weak and fragile
- **Increased durability** for golden instruments
- Golden Pickaxe now has **iron level**
- Golden instruments, weapons and armor are now **Gilded Iron and crafted on anvil** require Iron variation and Gold Ingot

## Minimalism
- Only **1 new item**
- Changes mostly done via mixins and loot tables
- Lightweight and highly compatible
- Vanilla-like feel and visuals

## Configuration
Partially configurable via **Cloth Config**

## Compatibility
Works great with:
- TreeCapitator-like mods
- Any tech mods (Create, Tech Reborn, AE2, etc.)
- Most exploration and worldgen mods
